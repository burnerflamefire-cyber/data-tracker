# Project Data Tracker

## Purpose
A modular web-based data tracking system for managing programme and project-level infrastructure information. 
Built for structured reporting, filtering, and future analytics expansion.

## Tech Stack
- Frontend: HTML + Vanilla JS (modular)
- Auth: Firebase Authentication (Anonymous)
- Database: Firebase Realtime Database
- Hosting: GitHub Pages
- Future CI/CD: Netlify (optional)

## Current Status
- Firebase authentication working (anonymous login)
- App loads successfully
- Base modular structure being introduced
- UI shell (navigation system) in development

## Core Goals
- Fast, lightweight data entry and tracking
- Clear separation between Programme and Project levels
- Expandable module-based architecture
- Government-grade UI styling (Western Cape blue theme)
- Fully browser-based, no install required
